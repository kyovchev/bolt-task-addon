# Firmware for Task Board v2023 with Bolt Task Addon

This folder contains Arduino and IDF firmwares for the Task Board v2023 with the Bolt Task Addon. They are compatible with the older end-of-life M5StickC Plus2 and the current M5Stack StickS3.

The Arduino firmware is very basic and shows how to interact with all of the sensors. The Arduino firmware is in the folder [./Arduino/TBv2023_Bolt_Task_Addon/](./Arduino/TBv2023_Bolt_Task_Addon/).

The IDF firmware is an extension of the original firmware. It just made compatible with the new sensors and the newer StickS3. The original version of the firmware can be found at [https://github.com/peterso/robotlearningblock/tree/main/idf/taskboard](https://github.com/peterso/robotlearningblock/tree/main/idf/taskboard). More information about the instalation procedure can be found in the folder [./IDF/](./IDF/).
